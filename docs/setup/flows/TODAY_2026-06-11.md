## Top 5 gotchas for today (the ones that will actually bite)

1. **Option labels must be the literal strings `Collision` / `Comp-Weather`** — CreateClaim maps those exact strings to choice integers.
2. **Multiple-choice may return an object, not text** — if CreateClaim gets a blank `lossType`/`injuryFlag`, bind `Topic.lossType.Value` instead.
3. **LogDecision takes `Global.claimGuid` (the GUID), never the CLM- number** — otherwise the audit row saves orphaned.
4. **Both injury branches must reach CreateClaim** — the Yes branch adds a message only; CreateClaim sits on the re-joined trunk.
5. **Test pane = draft, iframe = published** — after any topic edit, Publish again or the embed lags behind.
